# Operations Research Projects

A collection of practical Operations Research (OR) projects focused on solving real-world decision-making problems through optimization, simulation, heuristics, metaheuristics, and data analytics.

The objective of this repository is to demonstrate how Operations Research techniques can be applied to improve logistics, production, inventory management, scheduling, and other industrial engineering problems.

## Current Projects

### Vehicle Routing Optimization using Genetic Algorithms

A realistic logistics case based on Lima Metropolitana.

The project optimizes customer deliveries while considering operational constraints such as:

- Vehicle capacities
- Delivery time windows
- Customer priorities
- Incompatible cargo
- Maximum working hours
- Operational penalties

Outputs include:

- Route optimization
- Interactive maps
- Evolution GIF
- MP4 animation
- Delivery reports
- Convergence analysis
